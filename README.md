# Streamelements Overlays and widgets I made

## My overlays and widgets created in the Streamelements editor

The Streamelements editor allows you to create interactive and customizable overlays and widgets for you Twitch streams without needing to host them on your own server and without needing to handle the Twitch API connection.

It is also a way to being able to share these widgets and overlays to people who are not used to coding.

### How to add custom widgets

In the bottom-left corner of the editor, click the + icon, then hover on "static / custom" and select "custom widget".
Click the newly added custom widget.
On the left, you'll have menu items. Click on "Settings" then "open editor".
That is where you can play around with coding your own overlays/widgets.
